How to Integrate

Add this code to your Flutter app to start protecting your payments:

1. Add the dependency

dependencies: api_sentinel: ^1.0.0

2. Initialize the SDK

import 'package:api_sentinel/api_sentinel.dart'; final sentinel = ApiSentinel( apiKey: 'YOUR_API_KEY', baseUrl: 'https://95477e84-9911-4614-bb62-52ba832fcb90-00-1a973zog8mm4f.janeway.replit.dev:8080', );

3. Use in your payment flow

final result = await sentinel.processPayment( amount: 99.99, currency: 'USD', primaryGateway: 'stripe', fallbackGateway: 'paypal', metadata: {'orderId': '12345'}, ); if (result.success) { print('Payment recovered: \$${result.amount}'); }
